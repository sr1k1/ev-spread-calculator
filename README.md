EV Spread Calculator

Description:
This tool calculates the optimal EV spread for a Pokemon of your choice to counter another Pokemon of a given tier on Pokemon Showdown. This tool also gives you feedback on an EV spread of your choice and gives you a list of all the Pokemon it can reliably check/counter.

Dependencies:

- react-router for routing between the different available tools
- styled-components

How to install and run:

- Downloading the repo and running package.json should be all you need to install for the app!

Details on API connection:

- Except for the connection to the database (Airtable), all other API connections do not require any credentials to access. I am using PokeApi and SmogonApi for my information.
